# 🌤️ Weather Dashboard (天气仪表盘)

A radically simple, no-nonsense weather forecast web application powered by the OpenWeather API. 

## 💡 Why This Exists (项目初衷)

Let's be honest: checking the weather shouldn't require navigating a maze of ads or giving up your privacy. 

While there are countless weather apps out there, they often fall into two categories:
1. **The Bloated Official Apps:** Packed with splash-screen ads, irrelevant news feeds, and unnecessary features (especially common in mainland Chinese app markets).
2. **The "Free" Third-Party Apps:** Often lack transparency, leaving you wondering what data they are collecting in the background.

**MinimalWeather is built for the desktop dwellers and the open-tab warriors.** If your main focus is on your computer screen, you shouldn't have to break your workflow, find your phone, and open an app just to see if you need a jacket before stepping out. 

Just open the bookmark, type your city, get the weather, and go.

## ✨ Features (特性)

* **Zero Ads, Zero Distractions:** No pop-ups, no news feeds, no "premium" upgrades. Just the weather.
* **Desktop-First Efficiency:** Designed for quick access via your browser. Open it, check it, close it.
* **100% Transparent:** Fully open-source on GitHub. You can inspect the code yourself to ensure it's safe, secure, and purely functional.
* **Laser-Focused UI:** Input a city -> Get the current weather conditions. That's it. 

## 🛠️ Tech Stack (技术栈)

* HTML5 / CSS3 / JavaScript
* [OpenWeather API](https://openweathermap.org/) for real-time weather data
* Hosted on GitHub Pages
